# Modified-ForumGrandmaster-for-Discuz-
this is a modified Forum Grandmaster for Discuz! (Final Version) 论坛大师・Discuz!（全功能最终版）-- 修改版，
是tampermonkey的插件，我主要是适配right.com.cn恩山论坛。
因为回帖内容大都是无效的内容，应我的需求调整。

原作者
https://github.com/hishis/forum-grandmaster-for-discuz
源代码，他写的是MPL。
我也就跟随了，我也是MPL。


我就增加一下我需要的功能，有点破坏原来的主要内容。也产生了一些bug。

1. 修改删除了更新链接和原来脚本homepage信息。不为别的，怕误点了更新。

2. 我仅仅测试right.com.cn恩山论坛，别的没测试，不兼容我也没办法。
3. 我添加的主要内容
   a. 双栏浏览，这样大大提高效率，大部分论坛回帖都是无意义的内容，所以需要迅速浏览
   b. 隐藏了默认论坛的左侧边栏，就是作者信息图标和动态和积分等内容的，
   c. 把作者名字放到了有栏顶栏，这样就是谁-什么时间发表的。
   d. 自动翻页，不断获取下一页帖子，毕竟默认15贴一页，我浏览100页，点死人了的下一页。但是页数索引不更更新，你要自己算，翻到那一页了，点击下一页，可能是很久以前的了。
